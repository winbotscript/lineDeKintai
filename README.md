Line De 勤怠
====

## Overview

遅刻ぎりぎりのあなたにおすすめ！  
Lineでワンクリックするだけタイムカードを打刻できます。 

## Description
- 毎朝9時に打刻を催促
- 出社・退社処理をワンクリックで実施
- 定時になると煽りメッセージ

## Requirement
- Go 1.10 or later
- github.com/yuki9431/mongoHelper
- github.com/yuki9431/logger
- github.com/globalsign/mgo/bson
- github.com/line/line-bot-sdk-go/linebot/httphandler
